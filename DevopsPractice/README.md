# Practice Task 

## Instructions

Open a terminal and do the following:

1. Create a folder named `DevOpsPractice`
2. Change into that folder
3. Create another folder named `week01` and change into it
4. Create two files with any names you choose
5. Create two folders
6. List the contents of your current directory
7. Create an alias that copies folders (recursive copy)
8. List all existing aliases
9. Use your alias to copy one of your folders
10. List the directory again to confirm the copy worked
11. Remove the alias and verify it is removed (list aliases again)
12. Rename one of the files you created
13. Move the second file into one of the folders
